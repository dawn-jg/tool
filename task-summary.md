# Tooltip.cc 在线工具站

## 目标
构建一个免费在线工具网站，包含 7 大分类、55+ 工具页面，纯前端实现。

## 技术栈
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (图标)
- qrcode (二维码生成)
- 部署目标：Vercel + GitHub Pages

## 域名
- tooltip.cc (工具站主域名)
- qinheli.net (亲和力，主域名保留)

## 目录
- D:\tooltip.cc\

## 工具分类

### 1. 开发者工具（高频）
- JSON 格式化、正则测试、Base64 编解码、时间戳转换、SQL 格式化、YAML 转换、JWT 解码、哈希生成、UUID 生成、HTML 实体

### 2. 文本处理（中频）
- 文本对比、Markdown 编辑器、Case 转换、字数统计、Lorem Ipsum、Slug 生成、标点转换

### 3. 图片工具
- 图片 Base64、Favicon 生成、二维码生成、条形码生成、SVG 转 JSX、Exif 查看

### 4. 数据工具
- 密码生成、Cron 生成、进制转换

### 5. 生成器
- 占位图、假数据、渐变色、Meta 标签、Data URI

### 6. 验证器
- JSON 校验、邮箱验证、手机号验证、身份证验证、信用卡校验、URL 验证、颜色对比度

### 7. 实用工具
- 单位换算、时区转换、计时器、随机抽签、计算器、BMI 计算
